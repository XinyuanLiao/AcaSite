# AcaSite: Academic personal webSite
This is a simple but fancy academic personal page template based on HTML, CSS, and JavaScript. This website is static and can be hosted on the GitHub page or Netlify.

This is the demo website link: [https://vetiver.netlify.app/](https://vetiver.netlify.app/).

## Customize
It's easy to change the code in ```index.html``` by following the original code.
